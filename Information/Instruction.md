## 🚀 三步上手

**第一步：设置 AI 模型**  
点右上角侧栏 → 找到「模型」那一行，点右边的齿轮图标 ⚙  
填入你的 API Key（推荐用 **DeepSeek**，便宜好用；也支持 OpenAI、Claude）

**第二步：导入数据**  
点侧栏的「＋ 添加数据源」，支持：
- **Excel / CSV 文件**：直接拖进来
- **数据库**：MySQL、PostgreSQL 等（需要连接地址）
- **Google Sheets**：在线表格也能接入
- **网络 API**：从其他系统实时拉数据

**第三步：开始提问**  
在底部输入框直接说人话就行，比如：
- `这个月销售额多少？`
- `销量前十的产品是哪些？`
- `帮我做个销售趋势图`

## ⚡ 斜杠命令

输入 `/` 调起命令面板，可以快速触发特定功能：

### 分析 · 图表
| 命令 | 能做什么 |
|------|---------|
| `/chart`      | 自动选最合适的图表类型并生成 |
| `/sql`        | 直接运行数据查询，结果以表格展示 |
| `/decile`     | 把数据按高低分成十档，看各档占比 |
| `/tree`       | 用决策树找出影响结果最重要的因素 |
| `/kmeans`     | 自动给用户 / 商品等做分群分类 |
| `/logistic`   | 预测"会不会发生"（如流失、转化）的概率 |
| `/regression` | 分析一个指标与多个因素之间的关系 |

### 时间序列预测
| 命令 | 能做什么 |
|------|---------|
| `/arima`   | 经典时间序列预测，适合周期规律明显的数据 |
| `/sarima`  | 带季节性的预测，自动识别周 / 月 / 年周期 |
| `/var`     | 同时预测多个相互影响的指标 |
| `/prophet` | Meta 开源预测模型，对节假日和趋势变化更敏感 |
| `/gru`     | 深度学习预测，适合数据量大、规律复杂的场景 |

### 数据清洗
| 命令 | 能做什么 |
|------|---------|
| `/data`      | 一键查看数据质量：有多少缺失、分布是否正常 |
| `/inset`     | 自动填补缺失值（可选：填 0、均值或中位数） |
| `/winsorize` | 把极端异常值"压"回合理范围内 |
| `/trimming`  | 直接删掉范围外的异常行 |

### 导出
| 命令 | 能做什么 |
|------|---------|
| `/export`    | 把分析结果导出为 Excel 文件 |
| `/report`    | 生成完整的 Word 分析报告 |
| `/ppt`       | 生成一份麦肯锡风格的 PPT |
| `/dashboard` | 生成可以点击交互的销售看板网页 |

### 高级分析
| 命令 | 能做什么 |
|------|---------|
| `/screening` | 多条件数据筛选，快速缩小分析范围 |
| `/funnel`    | 漏斗分析，查看转化路径每一步的流失情况 |

### 系统 · 会话
| 命令 | 能做什么 |
|------|---------|
| `/status`  | 查看当前用的模型、数据源和已消耗的 Token 数 |
| `/new`     | 新建一个空白对话 |
| `/stop`    | 停止正在生成的回复 |
| `/compact` | 手动压缩对话上下文，减少 Token 用量 |
| `/clear`   | 清除当前对话记录 |
| `/session` | 管理已保存的对话（列表 / 加载 / 删除） |
| `/help`    | 查看可用命令说明 |
| `/plan`    | 为当前会话设置一条持久指令（Agent 每次回复时都会遵守） |
| `/teams`   | 打开分析团队面板，查看团队成员和沟通记录 |
| `/workspace` | 打开工作目录面板 |
| `/rewind`  | 打开文件历史 / 对话快照，支持时光回退 |
| `/jobs`    | 查看后台任务历史 |
| `/memory`  | 打开知识库面板 |
| `/mcp`     | 打开 MCP 服务器配置面板 |
| `/skill`   | 打开分析技能选择器 |

## 💡 使用技巧

### 直接说需求就好
不用想着要用哪个命令，描述你想知道什么，AI 会自己决定怎么分析、用什么图。比如：
> "帮我看看最近三个月各地区的销售变化，用折线图展示"

### 导出前先确认大纲
生成 PPT、Word 报告、看板时，AI 会先给你一份**大纲预览**，让你确认后再生成。你可以：
- 点 **✅ 确认生成** 直接出文件
- 点 **✏️ 修改大纲** 告诉 AI 改哪里（如"把第 3 页换成文字说明"）
- 点 **✕ 取消** 放弃本次生成

### 数据预览
右上角「数据预览」按钮可以直接看表格，按 sheet 切换，最多显示 100 行。

### 保存对话
侧栏底部「💾 保存」可以给当前对话命名，下次想继续分析直接点开，数据源会自动重连。

### 业务知识库
侧栏「🧠 业务知识库」可以告诉 AI 你们公司特有的计算规则和指标定义，比如"我们的 DAU 要去掉内部员工账号"，AI 以后分析时就会自动考虑这些规则。

### MCP 工具扩展
侧栏「MCP」可以接入外部工具（比如让 AI 帮你操作浏览器、读取本地文件等），有需要再单独配置。

### 分析团队（Teams）
面对复杂问题时，可以让 AI 自动组建一个轻量分析团队，把任务拆分给不同"角色"并行处理：
- 一个角色负责查数据，一个角色负责验证结论，一个角色负责撰写总结
- 结果会交叉检查，准确度更高
- 用 `/teams` 可以查看团队的分工和沟通记录

**开启方式**：侧边栏底部 **⚙ 设置** → 助手体验 → 打开 **Teams** 开关（默认关闭）。
> 💡 Teams 会消耗更多 Token，适合重要分析场景使用。

### 智能提问建议（Prompt Suggestion）
每次 AI 回复完成后，输入框会用浅色文字提示"你接下来可能想问的问题"：
- 直接按 **Tab** 或点击输入框即可采纳建议
- 输入任意文字会自动清除建议
- 不想要这个功能？在 **⚙ 设置** → 助手体验 → 关闭 **Prompt Suggestion** 开关

### 工作目录（Workspace）
除了上传 Excel / CSV，你还可以把本地文件夹"挂载"给 AI 当工作目录：
- AI 可以读取和编辑目录里的文件（最大 20 MiB）
- 分析产生的派生表可以持久化保存在工作目录的数据库中
- 支持**只读**和**可读写**两种权限
- 用 `/workspace` 查看已连接的工作目录，或在侧栏底部点击工作目录入口
- 文件修改会自动记录历史，出错了可以用 `/rewind` 回退

## 🛠 遇到问题？

| 现象 | 怎么解决 |
|------|---------|
| 提示"认证失败"或 401 | API Key 填错了，点侧栏 ⚙ 重新填入 |
| 提示请求太频繁（429） | 系统会自动重试，稍等一会儿，或换一个模型 |
| 图表生成失败 | AI 写的查询可能有问题，展开工具步骤看具体错误提示 |
| 数据源显示"已断开" | 刷新页面或新建会话，重新连接即可 |
| 对话越来越长、回复变慢 | 正常现象，AI 会自动整理上下文，不影响使用 |
