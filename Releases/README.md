# Releases

This folder contains the Windows installer for **Business Analysis Agent**.

## Latest Release

| File | Version | Size | Platform |
|---|---|---|---|
| `BusinessAnalyticsAgent_Setup.exe` | v5.1 | ~44 MB | Windows 10 / 11 64-bit |

## How to Install

1. Download `BusinessAnalyticsAgent_Setup.exe`
2. Run the installer and follow the on-screen steps
3. Launch **BusinessAnalyticsAgent** from the desktop shortcut or Start Menu
4. Configure your API Key in the settings panel

## Notes

- No Python installation required — the runtime is bundled
- On first launch the app configures itself automatically
- To uninstall, use **Add or Remove Programs** in Windows Settings
